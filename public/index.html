List of IgniteVR overlays.

Most overlays require ?client_name=USERNAME to retrieve the correct data.

Overlays require the OBS source to be set to 1920x1080 unless otherwise specified.

VRML

Echo Arena

  • /match_setup
    • Contains a list of upcoming matches. Click on one to set the team name and logo for the rest of the overlays.
    • Has an input field for client name.
  • /casterdesk Caster Desk
    • An overview of the current match. Contains previous matches by both teams and detailed game stats.
    • Has an input field for client name.
  • /group_recent_matches Group Recent Matches
    • A list of recent matches, with the ability to hide or split data from certain rounds. Used to verify data before it appears in the scoreboard overlays.
    • Parameter(s): client_name, series_name
  • /prematch_overlay
    • Shows a list of player names for both teams near the botton of the screen.
    • Parameter(s): client_name, series_name
  • /prematch_overlay_old
    • Team vs Team at the top center, left and right sidebars with team logos, mmr, division, and player names
    • Parameter(s): client_name, series_name
  • /midmatch_overlay
    • Team names and logos at the top of the screen
    • Parameter(s): client_name, series_name
  • /prematch_overlay_2
    • Team names, logos, division, mmr, and record at the bottom of the screen
    • Parameter(s): client_name, series_name
  • /most_recent_match
    • Old style player and team stats at the bottom of the screen. Team stats are on the sides.
    • Parameter(s): client_name, series_name, live
  • /all_recent_matches
    • List of recent rounds with time, caster, and scores
    • Parameter(s): series_name (optional: if not specified, shows all series)
  • /fastest_jousts
    • List of fastest joust time events recorded
    • Parameter(s): series_name (optional: if not specified, shows all series)
  • /scoreboard
    • New style player and team stats at the bottom of the screen. Team stats are on the bottom.
    • Parameter(s): client_name, series_name, live

Onward

  • /onward/match_setup
    • Contains a list of upcoming matches. Click on one to set the team name and logo for the rest of the overlays.
    • Has an input field for client name.
  • /onward/casterdesk
    • Overview for a matchup. Contains team info, previous matches for each team, map history, and previous direct matchups.
    • Has an input field for client name.
  • /onward/scoreboard
    • Overlay for scoreboard scene. Has team names, team logos, caster names/images, and analyst name.
    • Parameter(s): client_name
  • /onward/preshow
    • Overlay for scoreboard scene. Has team names, team logos, caster names/images, and analyst name.
    • Parameter(s): client_name
  • /onward/caster_1
    • Overlay that contains only the caster 1 name/image.
    • This overlay can be set to a smaller resolution and then positioned in OBS.
    • Parameter(s): client_name
  • /onward/caster_2
    • Overlay that contains only the caster 2 name/image.
    • This overlay can be set to a smaller resolution and then positioned in OBS.
    • Parameter(s): client_name
  • /onward/caster_3
    • Overlay that contains only the caster 3 name/image.
    • This overlay can be set to a smaller resolution and then positioned in OBS.
    • Parameter(s): client_name
  • /onward/analyst
    • Overlay that contains only the analyst name.
    • This overlay can be set to a smaller resolution and then positioned in OBS.
    • Parameter(s): client_name
  • /onward/home_team_logo
    • Overlay that contains only the home team logo.
    • This overlay can be set to a smaller resolution and then positioned in OBS.
    • Parameter(s): client_name
  • /onward/away_team_logo
    • Overlay that contains only the away team logo.
    • This overlay can be set to a smaller resolution and then positioned in OBS.
    • Parameter(s): client_name
  • /onward/home_team_name
    • Overlay that contains only the home team name.
    • This overlay can be set to a smaller resolution and then positioned in OBS.
    • Parameter(s): client_name
  • /onward/away_team_name
    • Overlay that contains only the away team name.
    • This overlay can be set to a smaller resolution and then positioned in OBS.
    • Parameter(s): client_name
  • /onward/home_team_roster
    • Overlay that contains only the home team player list.
    • This overlay can be set to a smaller resolution and then positioned in OBS.
    • Parameter(s): client_name
  • /onward/away_team_roster
    • Overlay that contains only the away team player list.
    • This overlay can be set to a smaller resolution and then positioned in OBS.
    • Parameter(s): client_name

CVRE

  • /match_awards
    • Shows a series of custom match awards
    • Parameter(s): client_name, series_name
  • /cvre_stats_blue
    • Shows player and team stats.
    • This overlay source should be set to 750x360 instead of 1920x1080 and then positionally adjusted with OBS
    • To increase the size even further, "body { font-size: 1.5em; }" can be used as a css override. The source resolution should also be changed. The default font-size is 1.5em
    • Parameter(s): client_name, series_name
  • /cvre_stats_orange
    • Shows player and team stats.
    • This overlay source should be set to 750x360 instead of 1920x1080 and then positionally adjusted with OBS
    • To increase the size even further, "body { font-size: 1.5em; }" can be used as a css override. The source resolution should also be changed. The default font-size is 1.5em
    • Parameter(s): client_name, series_name
